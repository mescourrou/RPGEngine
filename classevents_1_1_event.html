Detailed Description

template<typename... Args>
class events::Event< Args >

Handle subsciption to an event and triggering it.

See Event<void> for event without arguments

Example:

Event<int> event;
bool argCb = false;
auto cb = [&argCb] (int number) {
if (number == 3)
argCb = true;
};
event.subscribe(cb);
event.trigger(3);

With an object method, you need to use std::bin.

Event<std::string> event;
AClass o;
e2.subscribe(std::bind(&AClass::argMethod, &o, std::placeholders::_1));
e2.trigger("Hello");
