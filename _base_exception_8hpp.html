Create the exception class corresponding to the given class name.

To add codes, just add the list of ADD_EXCEPTION_CODE macro after the name of the class.

Exemple:

1 // Simple exception class
2 CREATE_EXCEPTION_CLASS(MySimpleClass)
3 
4 // With exceptions codes:
5 CREATE_EXCEPTION_CLASS(MyComplexClass,
6  ADD_EXCEPTION_CODE(CODE1) \
7  ADD_EXCEPTION_CODE(CODE2))

The exemple above will be processed into :

1 class MySimpleClassException : public BaseException
2 {
3 public:
4 MySimpleClassException(const std::string& w, const Errors& code = BaseException::UNKNOWN):
5  BaseException(w, code) {}
6 ~MySimpleClassException() override = default;
7 };
